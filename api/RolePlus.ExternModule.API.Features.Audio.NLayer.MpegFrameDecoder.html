    DecodeFrame(IMpegFrame, Single[], Int32)

    Decode the Mpeg frame into provided buffer. Result varies with different StereoMode:

    • For Both, sample data on both two channels will occur in turn (left first).
    • For LeftOnly and RightOnly, only data on specified channel will occur.
    • For DownmixToMono, two channels will be down-mixed into single channel.
